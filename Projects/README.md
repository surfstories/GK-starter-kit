# Projects

This is the home for everything you build with your assistant. **Every project gets its own folder**
here — `Projects/<name>/` — never loose files in the main folder.

Your first one, `my-digest/`, is created during onboarding.

A good habit: **one task = one new chat.** Start each new thing in a fresh chat; whatever it produces
lands in its own folder here. Over time this folder becomes the record of your work — and the more
that's here, the better your assistant knows how you work.
