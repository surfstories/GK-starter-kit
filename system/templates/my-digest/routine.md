# My Digest — the routine

Follow these steps when {{name}} says **"catch me up"** (or runs the `catch-me-up` skill).
Everything in {{name}}'s working language. This only **reads** — it never changes anything in Google.

1. **Look at recent Google files.** Use your Google Drive tools to fetch the ~8–10 most recently
   modified files (name, type, and when it changed). If Drive isn't connected, say so and offer to
   connect it (see the onboarding "Connect your Google" step). This is read-only.
2. **Add today's plan, if Calendar is on.** If Google Calendar is connected, also read **today's**
   events (time + title). If it isn't, skip this quietly.
3. **Write a short, friendly digest** to `digest-<YYYY-MM-DD>.md` in this folder, with:
   - **Today** — today's calendar events (if any), earliest first.
   - **Recently touched** — the recent files, grouped simply (e.g. Docs / Sheets / other), newest
     first, each a short line.
   - **One nudge** — a single suggestion for what to look at or do first.
4. **Show it** — paste the digest back in the chat and offer to open or summarize any one item.

Keep it warm and quick — this is a 1-minute "what's going on" catch-up, not a report. Nothing is
written to Google; only a local note in this folder.
