# Decisions log

A light, append-only record of how you like things done — kept so we don't lose the reasoning and
so patterns show up over time. Add new entries at the **bottom**; never rewrite or delete old ones
(a changed decision gets a *new* entry that supersedes the old).

**When to log:** any time you decide how something should be done and it might matter later — a
naming choice, a workflow, a "do X not Y", a preference. The assistant will offer to log these as
they come up; you can also just say "log this decision".

Entry template:
```
### YYYY-MM-DD — <short title>
- **Decision:** what you decided.
- **Why:** the reason.
```
